## Proyectos Destacados

### 1. Organizador Automático de Archivos (`/herramientas`)
Un script en Python diseñado para organizar el contenido de la carpeta de **Descargas**. Detecta de forma automática el sistema de archivos y clasifica los elementos en carpetas específicas:

* **Imágenes:** `.jpg`, `.png`, `.jpeg`
* **Videos:** `.mp4`, `.mov`, `.mkv`
* **Audios:** `.mp3`, `.wav`
* **Documentos:** `.pdf`, `.docx`, `.txt`, `.doc`
* **Software:** Instaladores `.exe` y `.msi`

**Tecnologías:** Python 3, librerías `os` y `shutil`.
